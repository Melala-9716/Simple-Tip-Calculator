💰 Tip Calculator

A simple and beginner-friendly Tip Calculator built using HTML, CSS, and JavaScript.
It helps users calculate the total bill amount including tip percentage.

🚀 Features
Enter bill amount
Enter tip percentage
Instant calculation of total
Simple and clean UI
Beginner-friendly JavaScript logic
🛠️ Built With
HTML
CSS
JavaScript (Vanilla JS)
🧮 Formula Used
Total = Bill + (Bill × Tip / 100)
📂 Project Structure
Tip-calculator/
 ├── index.html
 ├── style.css
 ├── script.js
 ├── README.md
💡 How It Works
User enters bill amount
User enters tip percentage
Click "Calculate"
JavaScript computes total
Result is displayed on screen
