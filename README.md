# Simple Tip Calculator 💰

A simple and beginner-friendly Tip Calculator built using HTML, CSS, and JavaScript.  
It helps users calculate the total bill amount including tip percentage.

---

## 🚀 Features
- Enter bill amount  
- Enter tip percentage  
- Select currency  
- Instant calculation of total  
- Dark/Light mode toggle  
- Simple and clean UI  
- Beginner-friendly JavaScript logic  

---

## 🛠️ Built With
- HTML  
- CSS  
- JavaScript (Vanilla JS)  

---

## 🧮 Formula Used
Total = Bill + (Bill × Tip / 100)

---

## 📂 Project Structure
Tip-calculator/
├── index.html

---

## 💡 How It Works
1. User enters bill amount  
2. User enters tip percentage  
3. User selects currency  
4. Click "Calculate"  
5. Result is displayed instantly
